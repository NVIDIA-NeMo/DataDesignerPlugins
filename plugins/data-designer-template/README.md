# data-designer-template

Template Data Designer plugin and reference implementation for plugin authors.

## Installation

```bash
pip install data-designer-template
```

## Usage

This plugin provides a `text-transform` column type for
[NeMo Data Designer](https://github.com/NVIDIA/NeMo-Data-Designer).
Once installed, the column type is automatically discovered by Data Designer.

For the full plugin authoring guide, see the
[main repository docs](https://gitlab-master.nvidia.com/etramel/data-designer-plugins/-/blob/main/docs/adding-a-plugin.md).
